SciTeX Seekbar Component

A flexible and accessible range slider component

1. Single-Handle Slider (Native HTML5)

Basic range slider using HTML5 native input. Works without JavaScript.

0 100
Value: 50
<div class="scitex-seekbar">
    <input type="range" class="scitex-seekbar-input"
           min="0" max="100" value="50">
</div>

2. Dual-Handle Slider (Basic)

Range slider with two handles for selecting a range. Requires JavaScript.

Range: 25 - 75
<div id="dual-basic"></div>

<script>
const seekbar = new ScitexSeekbar('#dual-basic', {
    min: 0,
    max: 100,
    valueMin: 25,
    valueMax: 75
});
</script>

3. Year Range Filter (Scholar App Style)

Publication year filter similar to the Scholar app implementation.

Years: 2000 - 2025
new ScitexSeekbar('#year-slider', {
    min: 1900,
    max: 2025,
    valueMin: 2000,
    valueMax: 2025,
    showValues: true,
    format: (value) => value.toString()
});

4. Citation Count Filter

Filter with formatted values (e.g., "1k", "10k+").

Citations: 0 - 12k
new ScitexSeekbar('#citations-slider', {
    min: 0,
    max: 12000,
    valueMin: 0,
    valueMax: 12000,
    step: 100,
    showValues: true,
    format: (value) => {
        if (value >= 10000) return (value/1000).toFixed(0) + 'k';
        if (value >= 1000) return (value/1000).toFixed(1) + 'k';
        return value.toString();
    }
});

5. Impact Factor Filter (Decimal Values)

Range slider with decimal step values for impact factor filtering.

Impact Factor: 0.0 - 50.0
new ScitexSeekbar('#impact-slider', {
    min: 0,
    max: 50,
    valueMin: 0,
    valueMax: 50,
    step: 0.1,
    showValues: true,
    format: (value) => value.toFixed(1)
});

6. Size Variants

Different sizes: small, default, and large.

7. Color Variants

Different color themes: primary, success, warning, danger.

8. Auto-Initialization (data-scitex-seekbar)

Automatically initializes from HTML attributes without writing JavaScript. 

<div data-scitex-seekbar="auto"
     data-min="-20"
     data-max="50"
     data-value-min="0"
     data-value-max="30"
     data-step="0.5">
</div>

9. API Methods

Programmatic control using JavaScript API.

Range: 30 - 70
const seekbar = new ScitexSeekbar('#api-slider', { ... });

// Set values
seekbar.setValues(40, 60);

// Get values
const values = seekbar.getValues();
console.log(values); // { min: 40, max: 60 }

// Reset
seekbar.reset();
