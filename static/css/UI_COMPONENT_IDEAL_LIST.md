UI Components Hierarchy
1. Foundation Components
1.1 Buttons

Primary Button - Main call-to-action
Secondary Button - Alternative actions
Tertiary/Text Button - Low emphasis actions
Danger Button - Destructive actions
Success Button - Positive confirmation
Loading Button - With spinner state
Button Group - Multiple related buttons
Icon Button - Icon-only or icon+text
Split Button - Button with dropdown

1.2 Typography

Headings (H1-H6)
Paragraph Text
Labels
Captions/Hints
Code/Monospace
Emphasis (bold, italic, underline)
Links

1.3 Spacing & Layout

Container
Spacer/Divider
Grid/Flex utilities

2.1 Input Fields

Text Input (single-line)
Textarea (multi-line)
Number Input
Email Input
Password Input
Search Input
URL Input
Tel Input

2.2 Selection Components

Checkbox (single & grouped)
Radio Button (single & grouped)
Toggle Switch
Select/Dropdown (single)
Multi-Select (multiple options)
Combobox/Autocomplete
Date Picker
Time Picker
Date Range Picker
Color Picker
File Upload

2.3 Form Layout

Form Group (label + input + hint)
Form Row (horizontal layout)
Form Section (grouped related fields)
Fieldset (grouped with legend)
Input Group (prepend/append additions)

2.4 Validation & Feedback

Error Message (inline)
Helper Text (hint/description)
Validation Icons (checkmark, error)
Input State (error, success, warning, disabled)

3.1 Table/Grid

Table (basic)
Data Grid (with sorting, filtering)
Table Row

Sortable Column Header

3.2 List Components

List (ordered/unordered)

Card (basic container)
Card Header/Body/Footer
Card with Image
Panel (bordered card)
Well (inset container)


4. Feedback & Alerts
4.1 Notifications

Alert (info, success, warning, error)
Toast/Notification (temporary)
Snackbar (bottom notification)
Banner (page-level announcement)

4.2 Progress & Status

Progress Bar (linear)
Progress Ring (circular)
Spinner/Loader (indeterminate)
Skeleton (content placeholder)
Status Badge (online/offline, etc.)

4.3 Confirmation & Interaction

Modal/Dialog
Confirmation Dialog
Alert Dialog
Tooltip
Popover


5. Navigation Components
5.1 Navigation Elements

Navbar/Header (top navigation)

Tabs (tabbed interface)

5.2 Navigation Links

Link (basic)

Disabled Link

6.2 Rich Content

Carousel/Slider (rotating content)
Lightbox (image viewer)
Video Player
Code Block (syntax highlighting)

6.3 Special Purpose

Empty State (no data message)
Loading State (skeleton/spinner)
Error State (error message + recovery)
