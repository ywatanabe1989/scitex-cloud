# Visualization Directory

This directory contains all visualization-related files for your research project.

## Directory Structure

- `figures/` - Final publication-quality figures
- `panels/` - Individual panel images that compose multi-panel figures
- `pinned/` - Important visualizations pinned for quick access
- `previews/` - Thumbnail previews of visualizations
- `metadata/` - Metadata and configuration for visualizations

## Supported File Types

The visualization workspace shows:
- Data files: `.csv`, `.tsv`, `.json`
- Images: `.png`, `.jpg`, `.jpeg`, `.svg`
- Documents: `.pdf`

## Getting Started

1. Create visualizations using the scripts in `/scripts/`
2. Save output images to `figures/` or `panels/`
3. Use the vis workspace (`/vis/`) to browse and manage your visualizations
