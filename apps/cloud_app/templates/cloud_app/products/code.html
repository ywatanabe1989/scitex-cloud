Advanced Scientific Computing with MNGS

SciTeX-PyTools is built on the MNGS (Monkeys) Python package, providing a comprehensive framework for neuroscience research and scientific computing. It offers powerful tools for data analysis, signal processing, and machine learning workflows.

Project Location: ~/proj/mngs_repo 
import mngs

# Load and preprocess neural data
data = mngs.io.load('neural_recording.h5')
filtered = mngs.dsp.bandpass(data, low=1, high=100)

# Apply machine learning pipeline
features = mngs.ml.extract_features(filtered)
results = mngs.ml.classify(features, model='svm')

# Visualize results
mngs.plt.plot_results(results, save_path='analysis.pdf')
