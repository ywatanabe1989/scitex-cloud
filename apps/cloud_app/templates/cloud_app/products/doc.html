SciTeX-Doc

LLM-friendly LaTeX compilation system for modern scientific writing

Revolutionizing Scientific Document Preparation

SciTeX-Doc makes LaTeX accessible to AI assistants and streamlines scientific writing. This modular LaTeX compilation system breaks down complex documents into manageable components, enabling AI-assisted writing, better organization, automated compilation, and seamless integration with research workflows.

Streamlined Writing Workflow

1. Structure Your Document

Use SciTeX-Doc templates to create a well-organized document structure with separate files for each section.

scitex init --template nature
scitex add-section methods
scitex add-figure fig1

2. Write with AI Assistance

AI can understand and edit individual sections without processing the entire document.

# AI edits methods.tex
scitex ai-edit methods \
  "Add statistical analysis section"

3. Compile and Review

Smart compilation catches errors early and provides helpful suggestions for fixes.

scitex compile
# Output: main.pdf
# Warnings: 2 undefined references

4. Collaborate and Revise

Track changes at the section level and manage reviewer comments efficiently.

scitex diff --revision 1
scitex add-response reviewer1.txt

Seamless Integration

SciTeX-Doc works perfectly with the entire SciTeX ecosystem. Get AI assistance for writing with SciTeX-Engine, automatically include generated figures with SciTeX-Viz, import citations from SciTeX Scholar, and include code snippets from SciTeX-PyTools with automatic formatting.
