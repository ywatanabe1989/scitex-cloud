Quick Installation

Get started with SciTeX Local in minutes using pip or conda:

Using pip

pip install scitex-local

Using conda

conda install -c scitex scitex-local

Using Docker

docker pull scitex/local:latest
docker run -p 8000:8000 scitex/local:latest

System Requirements

Minimum Requirements

  • Python 3.8 or higher
  • 8GB RAM
  • 10GB free disk space
  • Windows 10/11, macOS 10.14+, or Linux

Recommended Requirements

  • Python 3.10 or higher
  • 16GB RAM or more
  • 50GB free disk space
  • GPU support for ML tasks (optional)
