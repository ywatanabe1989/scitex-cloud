üì¶ What Gets Copied

When you click an element, the tool copies comprehensive debug information including:

  • Element tag, ID, classes, and CSS selector
  • All HTML attributes (href, src, data-*, aria-*, etc.)
  • Computed CSS styles (display, position, colors, fonts, etc.)
  • Dimensions and position on page
  • Text content preview (first 200 characters)
üí° Pro Tip: Paste the copied info into your conversation with Claude Code or other AI assistants for instant debugging help!

üîß Troubleshooting

  • Can't see overlay? Click the bookmark again - it might be toggled off
  • Copy failed? Some sites block clipboard access - check browser console for details
  • Keyboard shortcuts not working? Make sure you're not in an input field when pressing Alt+I
