Integrated environment for computational research workflows. From literature to publication with reproducible execution.

Repository API Docs

Research workflows typically span multiple disconnected tools: reference managers, text editors, analysis environments, plotting libraries. SciTeX provides integrated infrastructure connecting these components through shared workspaces with reproducible execution.

Technical Stack

Execution Infrastructure

  • Containers: Apptainer/Singularity (user-customizable)
  • Scheduling: SLURM (multi-user resource management)
  • Platform: Docker (web services)
  • Storage: Standard POSIX filesystems

Language Support

  • Any language: via containers
  • Common: Python, R, Julia
  • Scripts: Shell, Bash
