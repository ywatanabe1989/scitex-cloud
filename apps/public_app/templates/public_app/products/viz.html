Publication-Ready Scientific Visualizations

SciTeX-Viz, powered by SigMacro, provides sophisticated visualization tools specifically designed for scientific publications. Create stunning, interactive figures that bring your research data to life while maintaining the precision required for academic publishing.

Network Visualizations

Graph layouts, connectivity matrices, hierarchical structures, and flow diagrams

System Requirements

Platform Compatibility

  • Native Support: Windows 10/11
  • Linux/macOS: Via Apptainer container with Wine
  • Remote Option: Windows server for cloud processing

Key Dependencies

  • SigmaPlot: Version 14+ (license required)
  • Python: 3.8+ for scripting interface
  • Memory: 8GB RAM minimum, 16GB recommended
  • Storage: 2GB for software + data space

Container Availability

  • Apptainer Image: scitex/visualizer:latest
  • Includes: Wine, Python environment, automation scripts
  • Performance: ~90% of native Windows speed
  • Alternative: Remote Windows processing available

Licensing

  • SciTeX-Viz Scripts: MIT License
  • SigmaPlot: Commercial license required
  • Academic Pricing: Available through Systat Software
  • Future: Open-source alternative in development
Note: SigmaPlot is currently required for full functionality. We are developing an open-source alternative to provide Linux-native visualization capabilities without proprietary dependencies.

Seamless Integration

SciTeX-Viz integrates perfectly with other SciTeX tools, automatically generating visualizations from your analysis code and embedding them directly into your manuscripts with proper formatting and captions.
