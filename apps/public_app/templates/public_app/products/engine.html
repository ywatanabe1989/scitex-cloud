SciTeX Engine

AI-powered research assistant integrated directly into Emacs

Your AI Research Companion in Emacs

SciTeX Engine brings advanced AI capabilities directly into your Emacs workflow. Seamlessly integrating with your existing setup, it provides intelligent assistance for code generation, scientific document writing, literature review, data analysis, and research methodology suggestions.

Project Location: ~/.emacs.d/lisp/emacs-claude-code 
;; Example: Using SciTeX Engine in Emacs
(require 'scitex-engine)

;; Generate analysis code
(scitex-engine-generate
  "Create a Python function to analyze
   time-series data with FFT")

;; Get research suggestions
(scitex-engine-suggest-methods
  "neural signal processing")

Integrated Research Workflow

SciTeX Engine enhances every stage of your research process:

1. Literature Review

Summarize papers directly in Emacs, extract key findings, generate citation-ready notes, and compare multiple papers side-by-side.

2. Experiment Design

Generate experimental protocols, suggest statistical power calculations, create data collection templates, and design control conditions.

3. Data Analysis

Generate analysis scripts in Python/R, suggest appropriate statistical tests, create visualization code, and debug existing code.

4. Paper Writing

Draft sections based on results, format for target journals, generate response letters, and check citation consistency.

Easy Installation

SciTeX Engine is designed to integrate seamlessly with your Emacs configuration:

;; Add to your Emacs init file
(add-to-list 'load-path "~/.emacs.d/lisp/emacs-claude-code")
(require 'scitex-engine)

;; Configure API key
(setq scitex-engine-api-key "your-api-key")

;; Optional: Set custom keybindings
(global-set-key (kbd "C-c s g") 'scitex-engine-generate)
(global-set-key (kbd "C-c s e") 'scitex-engine-explain)
(global-set-key (kbd "C-c s r") 'scitex-engine-refactor)

Supported Modes

Works with Python, LaTeX/AUCTeX, Org, R/ESS, Markdown, and many more Emacs modes.
