Zotero API Key Setup

How to get your Zotero API Key:
  1. Go to Zotero API Keys page
  2. Log in to your Zotero account
  3. Create a new API key with the following permissions:
    • Allow library access
    • Allow write access (if you want to sync changes back)
    • Allow group access (if you want to sync group libraries)
  4. Copy the generated API key
  5. Paste it in the form below
