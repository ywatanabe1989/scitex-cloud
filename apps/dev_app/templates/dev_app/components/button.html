Button

Versatile button component with multiple variants, sizes, and states

{% include 'dev_app/design_partial/sidebar.html' %}

Overview

Buttons are interactive elements that trigger actions. They come in multiple variants (primary, secondary, outline, danger) and states (default, hover, active, disabled).

Category: Form and Selection

Dark Mode Support: Yes

Component Demo

Variants

  • btn-primary - Primary action button
  • btn-secondary - Secondary action button
  • btn-outline - Outline button with no fill
  • btn-danger - Destructive action button

States

  • default - Normal button state
  • hover - Hover state with color change
  • active - Pressed state
  • disabled - Disabled and cannot be clicked

HTML Example

<button class="btn btn-primary">Primary Button</button>
<button class="btn btn-secondary">Secondary Button</button>
<button class="btn btn-danger" disabled>Disabled</button>
