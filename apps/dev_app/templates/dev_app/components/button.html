Overview

Buttons are interactive elements that trigger actions. They come in multiple variants (primary, secondary, outline, danger) and states (default, hover, active, disabled).

Category: Form and Selection

Dark Mode Support: Yes

Production Classes: .btn, .btn-primary, .btn-secondary, .btn-danger

Variants

  • .btn-primary - Primary action button (main CTA)
  • .btn-secondary - Secondary action button
  • .btn-accent - Accent button
  • .btn-success - Success/confirmation button (green)
  • .btn-warning - Warning button (yellow)
  • .btn-danger - Destructive action button (red)
  • .btn-info - Informational button (blue)
  • .btn-dark - Dark button
  • .btn-outline-primary - Outlined primary button
  • .btn-outline-secondary - Outlined secondary button
  • .btn-outline-accent - Outlined accent button
  • .btn-text - Text-only button (tertiary/low emphasis)

Sizes

  • .btn-sm - Small button size
  • .btn - Default button size
  • .btn-lg - Large button size

States

  • default - Normal interactive state
  • hover - Highlighted on mouse over
  • focus - Highlighted when focused (keyboard navigation)
  • disabled - Non-interactive disabled state (opacity reduced)
  • loading - Shows spinner, non-interactive
