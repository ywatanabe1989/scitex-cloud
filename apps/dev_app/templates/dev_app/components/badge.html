Badge

Small labels for categorization and status indication

Overview

Category: Status & Labels

Dark mode: Supported

Badges are compact labels used to display status, count, or category information. They can be used in various colors to indicate different states or meanings.

Component Demo

Primary Secondary Success Warning Danger Neutral Light Outline

Variants & States

  • Primary: Default badge for general use
  • Secondary: Lighter blue variant
  • Success: Green badge for positive states
  • Warning: Orange badge for warnings
  • Danger: Red badge for errors/danger
  • Neutral: Gray badge for neutral states
  • Light: Light background with border
  • Outline: Transparent with colored border

HTML Code Example

<!-- Primary Badge -->
<span class="badge badge-primary">Primary</span>

<!-- Success Badge -->
<span class="badge badge-success">Success</span>

<!-- Danger Badge -->
<span class="badge badge-danger">Danger</span>

<!-- Outline Badge -->
<span class="badge badge-outline">Outline</span>

<!-- With Count -->
<span class="badge badge-primary">5</span>
