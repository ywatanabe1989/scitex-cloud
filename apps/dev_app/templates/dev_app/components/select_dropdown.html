Select Dropdown

Dropdown select component for choosing from multiple options

{% include 'dev_app/design_partial/sidebar.html' %}

Overview

Select dropdowns provide a space-efficient way for users to choose from a list of options. They collapse the list until activated.

Category: Form and Selection

Dark Mode Support: Yes

Component Demo

States

  • default - Collapsed dropdown with default selection
  • open - Expanded showing all options
  • focus - Focused with blue border
  • disabled - Disabled and cannot be interacted with

HTML Example

<select>
  <option value="">Select an option</option>
  <option value="1">Option 1</option>
  <option value="2">Option 2</option>
</select>
