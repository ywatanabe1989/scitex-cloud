Select Dropdown

Dropdown select component for choosing from multiple options

{% include 'dev_app/design_partial/sidebar.html' %}

Overview

Select dropdowns provide a space-efficient way for users to choose from a list of options. They collapse the list until activated.

Category: Form and Selection

Dark Mode Support: Yes

Production Class: .form-select

Component Demo

Looks good!
Please select an option.

States

  • default - Collapsed dropdown with default selection
  • open - Expanded showing all options
  • focus - Focused with highlighted border
  • disabled - Disabled and cannot be interacted with
  • valid - Valid selection with green border
  • invalid - Invalid selection with red border

Sizes

  • .form-select-sm - Small select dropdown
  • .form-select - Default size
  • .form-select-lg - Large select dropdown

HTML Example

<div class="form-group">
  <label class="form-label" for="country">Country</label>
  <select id="country" class="form-select">
    <option value="">Select a country...</option>
    <option value="us">United States</option>
    <option value="uk">United Kingdom</option>
    <option value="ca">Canada</option>
  </select>
</div>

CSS Classes

Class Description
.form-select Base select dropdown class
.form-select-sm Small size variant
.form-select-lg Large size variant
.is-valid Valid state
.is-invalid Invalid state
.form-group Container for label + select
.form-label Label styling
