Breadcrumb

Breadcrumb navigation component for showing the current page location

{% include 'dev_app/design_partial/sidebar.html' %}

Overview

Breadcrumbs show the current page location within a website's hierarchy. They provide a trail of links back to the homepage and improve navigation usability.

Category: Navigation

Dark Mode Support: Yes

Component Demo

Home / Projects / SciTeX / Design System
Home / Scholar / Libraries / Machine Learning Papers

States

  • default - Normal breadcrumb trail
  • active - Current page (last item, not a link)
  • hover - Breadcrumb link on hover underlines

HTML Example

<nav class="breadcrumb">
  <a href="/">Home</a>
  <span>/</span>
  <a href="/projects">Projects</a>
  <span>/</span>
  <span>Current Page</span>
</nav>
