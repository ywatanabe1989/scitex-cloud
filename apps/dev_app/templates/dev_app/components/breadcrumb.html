Breadcrumb

Breadcrumb navigation component for showing the current page location

{% include 'dev_app/design_partial/sidebar.html' %}

Overview

Breadcrumbs show the current page location within a website's hierarchy. They provide a trail of links back to the homepage and improve navigation usability.

Category: Navigation

Dark Mode Support: Yes

Production Classes: .breadcrumb, .breadcrumb-item, .breadcrumb-separator

States

  • default - Normal breadcrumb link (clickable)
  • active - Current page (last item, not a link, has .active class)
  • hover - Breadcrumb link on hover (background color change)

Variants

  • .breadcrumb - Container for breadcrumb navigation
  • .breadcrumb-item - Individual breadcrumb item (link or text)
  • .breadcrumb-item.active - Current page item (non-clickable)
  • .breadcrumb-separator - Visual separator between items (/, ›, etc.)

HTML Example

<nav class="breadcrumb">
  <a href="/" class="breadcrumb-item">Home</a>
  <span class="breadcrumb-separator">/</span>
  <a href="/projects" class="breadcrumb-item">Projects</a>
  <span class="breadcrumb-separator">/</span>
  <span class="breadcrumb-item active">Current Page</span>
</nav>

Alternative Separator Example

<nav class="breadcrumb">
  <a href="/" class="breadcrumb-item">Home</a>
  <span class="breadcrumb-separator">›</span>
  <a href="/docs" class="breadcrumb-item">Docs</a>
  <span class="breadcrumb-separator">›</span>
  <span class="breadcrumb-item active">Getting Started</span>
</nav>

CSS Classes

Class Description
.breadcrumb Container for breadcrumb navigation with flex layout
.breadcrumb-item Individual breadcrumb item (link or text)
.breadcrumb-item.active Current page item with bold text and surface background
.breadcrumb-separator Visual separator between items (/, ›, etc.)

Responsive Behavior

On mobile devices (<768px):

  • Font size is reduced for better fit
  • Horizontal scrolling is enabled if breadcrumb is too long
  • Custom scrollbar styling for better appearance
