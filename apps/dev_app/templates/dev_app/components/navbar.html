Navbar

Navigation bar component for top-level site navigation and branding

{% include 'dev_app/design_partial/sidebar.html' %}

Overview

Navigation bars are typically placed at the top of pages and contain the site logo/branding on the left and main navigation links on the right.

Category: Navigation Components

Dark Mode Support: Yes

Production Classes: .navbar, .navbar-brand, .brand-logo, .navbar-nav, .nav-item, .nav-link

Elements

  • .navbar - Main navbar container with flex layout
  • .navbar-brand - Brand/logo container area
  • .brand-logo - Logo link with large, bold text
  • .navbar-nav - Navigation links container (flex list)
  • .nav-item - Individual navigation item wrapper
  • .nav-link - Navigation link with hover and active states
  • .navbar-toggler - Mobile menu toggle button
  • .navbar-collapse - Collapsible navbar content on mobile

States

  • default - Normal navigation state
  • hover - Links change color and get background on hover
  • active - Current page link with distinct styling (.nav-link.active)
  • collapsed - Mobile collapsed state (hidden menu)
  • expanded - Mobile expanded state (visible menu with .show class)

Responsive Behavior

  • On desktop (>768px): Horizontal layout with inline navigation links
  • On mobile (≤768px): Toggle button appears, menu collapses vertically
  • Mobile menu expands full-width when toggle is clicked
  • Navigation items stack vertically on mobile for easy touch access

HTML Example

<!-- Basic Navbar -->
<nav class="navbar">
  <div class="navbar-brand">
    <a href="/" class="brand-logo">SciTeX</a>
  </div>
  <ul class="navbar-nav">
    <li class="nav-item"><a href="#" class="nav-link active">Home</a></li>
    <li class="nav-item"><a href="#" class="nav-link">About</a></li>
    <li class="nav-item"><a href="#" class="nav-link">Contact</a></li>
  </ul>
</nav>

<!-- Navbar with Mobile Toggle -->
<nav class="navbar">
  <div class="navbar-brand">
    <a href="/" class="brand-logo">SciTeX Cloud</a>
  </div>
  <button class="navbar-toggler" onclick="toggleMenu()">☰</button>
  <div class="navbar-collapse">
    <ul class="navbar-nav">
      <li class="nav-item"><a href="#" class="nav-link">Dashboard</a></li>
      <li class="nav-item"><a href="#" class="nav-link">Projects</a></li>
      <li class="nav-item"><a href="#" class="nav-link">Settings</a></li>
    </ul>
  </div>
</nav>

<script>
function toggleMenu() {
  document.querySelector('.navbar-collapse').classList.toggle('show');
}
</script>

.navbar-collapse Collapsible navbar content on mobile
.navbar-collapse.show Expanded mobile menu state

Dark Mode Support

The navbar automatically adapts to dark mode with the [data-theme="dark"] attribute:

  • Background changes to dark color (scitex-color-01-light)
  • Text colors adjust for proper contrast
  • Hover and active states use appropriate dark theme colors
  • Border colors adapt to maintain visibility
