Segmented Radio Control

Toggle buttons organized in a segmented control for mutually exclusive selections

{% include 'dev_app/design_partial/sidebar.html' %}

Overview

Segmented controls allow users to select one option from a group of mutually exclusive choices. They are visually grouped together and provide clear feedback about which option is selected.

Category: Form and Selection

Dark Mode Support: Yes

States

  • selected - Active/selected state with highlight
  • hover - Hover state with background change

HTML Example

<div class="segmented-control">
  <input type="radio" id="opt-small" name="option" checked>
  <label for="opt-small">Small</label>
  <input type="radio" id="opt-large" name="option">
  <label for="opt-large">Large</label>
