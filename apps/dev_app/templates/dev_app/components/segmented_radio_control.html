Segmented Radio Control

Toggle buttons organized in a segmented control for mutually exclusive selections

{% include 'dev_app/design_partial/sidebar.html' %}

Overview

Segmented radio controls allow users to select one option from a group of mutually exclusive choices. They feature circular radio indicators (iOS-style) and are visually grouped together in a unified control.

Category: Form and Selection

Dark Mode Support: Yes

Production Classes: .scitex-radio, .scitex-radio-label, .scitex-radio-group

States

  • default - Unchecked with empty circle indicator
  • checked - Circle filled with primary color and inner dot
  • hover - Background highlight and border color change on hover
  • mobile - Stacks vertically on mobile devices (<768px)

Size Variants

  • .scitex-radio-group - Default size with standard padding
  • .scitex-radio-group-sm - Smaller size variant with reduced padding and font size

HTML Example

<div class="scitex-radio-group">
  <input type="radio" id="all" name="sources" class="scitex-radio" checked>
  <label for="all" class="scitex-radio-label">All Sources</label>
  <input type="radio" id="pubmed" name="sources" class="scitex-radio">
  <label for="pubmed" class="scitex-radio-label">PubMed</label>
  <input type="radio" id="scholar" name="sources" class="scitex-radio">
  <label for="scholar" class="scitex-radio-label">Scholar</label>
</div>

<!-- Small Size Variant -->
<div class="scitex-radio-group scitex-radio-group-sm">
  <input type="radio" id="opt1" name="small" class="scitex-radio" checked>
  <label for="opt1" class="scitex-radio-label">Option 1</label>
  <input type="radio" id="opt2" name="small" class="scitex-radio">
  <label for="opt2" class="scitex-radio-label">Option 2</label>
</div>

CSS Classes

Class Description
.scitex-radio Hidden native radio input (visually hidden but accessible)
.scitex-radio-label Button-style label with circular radio indicator
.scitex-radio-group Container creating segmented control layout
.scitex-radio-group-sm Small size variant modifier

Accessibility

  • Native radio inputs remain in the DOM for keyboard navigation and screen readers
  • Labels are properly associated with inputs via for and id attributes
  • Circular indicators provide visual feedback of selection state
  • Radio buttons are grouped by name attribute for proper mutual exclusivity

Responsive Behavior

On mobile devices (screens below 768px), the segmented control automatically stacks vertically for better touch targets and readability.
