Overview

Checkboxes allow users to select multiple options from a list. The SciTeX design system provides custom-styled checkboxes with full theme support.

Category: Form and Selection

Dark Mode Support: Yes

Production Class: .scitex-checkbox, .scitex-checkbox-label

States

  • default - Unchecked state with border
  • checked - Checked with white checkmark icon on primary background
  • hover - Background highlight on label hover
  • disabled - Reduced opacity (not yet fully implemented)

Variants

  • .scitex-checkbox - Hidden native checkbox input
  • .scitex-checkbox-label - Custom styled label with visual checkbox
  • .scitex-checkbox-wrapper - Optional wrapper for positioning

Design Notes

Visual Design

  • The custom checkbox is created using the ::before pseudo-element on the label
  • Checked state uses an SVG checkmark icon embedded in CSS
  • Hover effect provides subtle background highlighting
  • Dark mode changes colors to bluish theme using SciTeX color variables

Implementation Status

  • Disabled state is partially implemented (opacity reduction on input)
  • Consider adding visual disabled styles to the label and ::before element
  • May need additional ARIA attributes for accessibility

Accessibility

  • Always associate labels with inputs using the for attribute
  • Provide clear, descriptive label text
  • Group related checkboxes with a visible heading or legend
  • Consider adding descriptions for complex options
  • Ensure sufficient color contrast for both light and dark themes
