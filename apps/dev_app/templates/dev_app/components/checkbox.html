Checkbox

Multiple selection input component

Overview

Category: Form & Selection

Dark mode: Supported

Checkboxes allow users to select multiple options from a list. They can be used individually or in groups.

Component Demo

Subscribe to Notifications
Disabled Options
Features
Enable dark theme for comfortable viewing
Automatically save your work in progress

Variants & States

  • Unchecked: Default state, ready for selection
  • Checked: Selected state with checkmark
  • Disabled: Unavailable for interaction (can be checked or unchecked)
  • With Label: Checkbox paired with descriptive text
  • In Group: Multiple checkboxes for multi-select

HTML Code Example

<!-- Single Checkbox -->
<label class="checkbox-label">
  <input type="checkbox">
  <span>Accept terms and conditions</span>
</label>

<!-- Checkbox Group -->
<div class="checkbox-group">
  <div class="checkbox-group-label">Preferences</div>
  <label class="checkbox-label">
    <input type="checkbox" checked>
    <span>Option 1</span>
  </label>
  <label class="checkbox-label">
    <input type="checkbox">
    <span>Option 2</span>
  </label>
</div>

<!-- Disabled Checkbox -->
<label class="checkbox-label">
  <input type="checkbox" disabled>
  <span>Unavailable option</span>
</label>
