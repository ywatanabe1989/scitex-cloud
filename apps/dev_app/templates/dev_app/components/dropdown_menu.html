Dropdown Menu

Dropdown menu component for displaying related actions or navigation options

{% include 'dev_app/design_partial/sidebar.html' %}

Overview

Dropdown menus present a list of actions or options that appear when triggered. They are useful for hiding secondary actions and keeping interfaces clean.

Category: Navigation

Dark Mode Support: Yes

Component Demo

States

  • default - Collapsed dropdown button
  • open - Menu is visible with all options
  • hover - Menu items highlight on hover
  • focused - Button is focused with blue border

HTML Example

<div class="dropdown-container">
  <button class="dropdown-button">Actions ▼</button>
  <div class="dropdown-menu">
    <a href="#edit">Edit</a>
    <a href="#delete">Delete</a>
  </div>
</div>
