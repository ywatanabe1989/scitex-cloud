Dropdown Menu

Dropdown menu component for displaying related actions or navigation options

{% include 'dev_app/design_partial/sidebar.html' %}

Overview

Dropdown menus present a list of actions or options that appear when triggered. They are useful for hiding secondary actions and keeping interfaces clean.

Category: Navigation

Dark Mode Support: Yes

Production Classes: .dropdown, .dropdown-toggle, .dropdown-menu, .dropdown-item

Component Demo

Actions
Edit Duplicate Share
Delete
File Menu
New Open Save
Export
User Menu
Profile Settings
Logout

States

  • closed - Menu hidden (default state)
  • open - Menu is visible with all options (.show class)
  • hover - Menu items highlight on hover with background change
  • mobile - Simplified mobile view (<768px)
  • desktop - Hover-triggered view (>768px)

Variants

  • .dropdown - Base container for dropdown component
  • .dropdown-toggle - Clickable trigger element with caret
  • .dropdown-menu - Hidden menu that appears on hover/click
  • .dropdown-menu-end - Right-aligned dropdown menu
  • .dropdown-item - Individual menu item link
  • .dropdown-divider - Horizontal divider between items

HTML Example

<div class="dropdown">
  <a href="#" class="dropdown-toggle">Actions</a>
  <div class="dropdown-menu show">
    <a href="#edit" class="dropdown-item">Edit</a>
    <a href="#delete" class="dropdown-item">Delete</a>
    <div class="dropdown-divider"></div>
    <a href="#share" class="dropdown-item">Share</a>
  </div>
</div>

Right-Aligned Example

<div class="dropdown">
  <a href="#" class="dropdown-toggle">User Menu</a>
  <div class="dropdown-menu dropdown-menu-end">
    <a href="#profile" class="dropdown-item">Profile</a>
    <a href="#settings" class="dropdown-item">Settings</a>
    <div class="dropdown-divider"></div>
    <a href="#logout" class="dropdown-item">Logout</a>
  </div>
</div>

.dropdown-menu.show Visible state for dropdown menu
