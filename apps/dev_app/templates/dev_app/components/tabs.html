Tabs

Tabbed interface for organizing content into separate views

{% include 'dev_app/design_partial/sidebar.html' %}

Overview

Tabs organize content into separate views within the same context. Users can switch between tabs to access different content categories.

Category: Navigation Components

Dark Mode Support: Yes

Production Classes: .tabs, .tab-item, .tab-content, .tab-pane

States

  • default - Inactive tab with secondary text color
  • active - Currently selected tab with primary text and border
  • hover - Hover state with highlighted border and text
  • disabled - Non-interactive disabled tab with reduced opacity

HTML Example

<div class="tabs">
  <button class="tab-item active">Overview</button>
  <button class="tab-item">Details</button>
  <button class="tab-item">Settings</button>
  <button class="tab-item" disabled>Locked</button>
</div>

<div class="tab-content">
  <div class="tab-pane active">
    <h3>Overview Content</h3>
    <p>Overview content here</p>
  </div>
  <div class="tab-pane">
    <h3>Details Content</h3>
    <p>Details content here</p>
  </div>
  <div class="tab-pane">
    <h3>Settings Content</h3>
    <p>Settings content here</p>
  </div>
</div>

CSS Classes

Class Description
.tabs Tab navigation container
.tab-item Individual tab button
.tab-item.active Active/selected tab state
.tab-item:disabled Disabled tab state
.tab-content Content container for tab panels
.tab-pane Individual tab panel content
.tab-pane.active Visible tab panel
