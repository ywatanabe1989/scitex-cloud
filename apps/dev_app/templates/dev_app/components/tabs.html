Tabs

Navigate between related sections of content

Overview

Category: Navigation

Tabs organize content into separate views within the same context. Users can switch between tabs to access different content categories.

Pills Style

This is an alternative tab style using pill-shaped buttons. This style works well for smaller tab sets or when you want a more modern appearance.

Alternative Style

Pills provide a different visual approach compared to the underline style above. Both are equally valid depending on your design needs.

Customizable

Tabs can be fully customized to match your application's design system. Colors, spacing, and interaction patterns can all be adjusted.

Variants & States

  • Standard Tabs: Underline style with border indicator
  • Pills Style: Button-like tabs with background fill
  • Active State: Currently selected tab is highlighted
  • Hover State: Visual feedback on tab hover
  • Disabled State: Inactive tabs can be disabled

HTML Code Example

<!-- Standard Tabs -->
<div class="tabs-nav">
  <div class="tabs-nav-item">
    <a class="tabs-nav-link active">Tab 1</a>
  </div>
  <div class="tabs-nav-item">
    <a class="tabs-nav-link">Tab 2</a>
  </div>
</div>

<div class="tabs-content">
  <div id="tab1" class="tabs-panel active">
    <h3>Tab 1 Content</h3>
    <p>Content for tab 1</p>
  </div>
  <div id="tab2" class="tabs-panel">
    <h3>Tab 2 Content</h3>
    <p>Content for tab 2</p>
  </div>
</div>

<!-- Pills Style -->
<div class="tabs-nav tabs-pills">
  <div class="tabs-nav-item">
    <a class="tabs-nav-link active">Active</a>
  </div>
  <div class="tabs-nav-item">
    <a class="tabs-nav-link">Inactive</a>
  </div>
</div>
