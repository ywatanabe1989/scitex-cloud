Toggle Button (Checkbox)

Button-style checkbox toggles with checkbox square indicator for on/off states

{% include 'dev_app/design_partial/sidebar.html' %}

Overview

Button-style checkbox toggles with checkbox square indicator for on/off states. Distinct from simple checkboxes by having a button-like container with borders.

Category: Form and Selection

Dark Mode Support: Yes

Component Demo

States

  • off - Unchecked state, checkbox square is empty
  • on - Checked state, checkbox square shows checkmark
  • hover - Background color changes on hover
  • disabled - Disabled state with reduced opacity

HTML Example

<label class="toggle-checkbox">
  <input type="checkbox" checked>
  <span>Option 1</span>
</label>
