Toggle Button (Checkbox)

Button-style checkbox toggles with checkbox square indicator for on/off states

{% include 'dev_app/design_partial/sidebar.html' %}

Overview

Button-style checkbox toggles with checkbox square indicator for on/off states. Distinct from simple checkboxes by having a button-like container with borders.

Category: Form and Selection

Dark Mode Support: Yes

Production Classes: .scitex-toggle, .scitex-toggle-label, .scitex-toggle-wrapper

Component Demo

States

  • off - Unchecked state, checkbox square is empty with border
  • on - Checked state, checkbox square filled with checkmark (✓)
  • hover-off - Border color changes when hovering over unchecked toggle
  • hover-on - Background color changes when hovering over checked toggle
  • disabled - Non-interactive state (can be checked or unchecked)

HTML Example

<!-- Basic Toggle -->
<div class="scitex-toggle-wrapper">
  <input type="checkbox" id="feature1" class="scitex-toggle">
  <label for="feature1" class="scitex-toggle-label">Enable Feature</label>
</div>

<!-- Checked Toggle -->
<div class="scitex-toggle-wrapper">
  <input type="checkbox" id="feature2" class="scitex-toggle" checked>
  <label for="feature2" class="scitex-toggle-label">Feature Enabled</label>
</div>

<!-- Disabled Toggle -->
<div class="scitex-toggle-wrapper">
  <input type="checkbox" id="feature3" class="scitex-toggle" disabled>
  <label for="feature3" class="scitex-toggle-label">Disabled Feature</label>
</div>

CSS Classes

Class Description
.scitex-toggle-wrapper Optional wrapper for positioning
.scitex-toggle Hidden checkbox input (controls toggle state)
.scitex-toggle-label Button-style toggle label with checkbox square
.scitex-toggle:checked Checked state selector
.scitex-toggle:disabled Disabled state selector

Usage Notes

  • Always use a unique id for the checkbox input
  • Match the for attribute of the label to the checkbox id
  • The wrapper is optional but helps with positioning
  • Checkmark (✓) automatically appears when checked
  • Use disabled attribute to prevent interaction
