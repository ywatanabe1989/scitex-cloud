Overview

Sidebar navigation provides a vertical menu for primary site navigation. It offers a persistent way for users to navigate between major sections.

Category: Navigation

Dark Mode Support: Yes

Component Demo

Home Projects Scholar Code Visualization Writer Settings

States

  • default - Normal sidebar link
  • hover - Link highlights with left border accent
  • active - Current page with blue accent and highlight

HTML Example

<nav class="sidebar-nav">
  <a href="#home">Home</a>
  <a href="#projects" class="active">Projects</a>
  <a href="#settings">Settings</a>
</nav>
