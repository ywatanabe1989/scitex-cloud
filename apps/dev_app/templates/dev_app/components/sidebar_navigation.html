Sidebar Navigation

Sidebar navigation component for main site navigation

{% include 'dev_app/design_partial/sidebar.html' %}

Overview

Sidebar navigation provides a vertical menu for primary site navigation. It offers a persistent way for users to navigate between major sections with sticky positioning and smooth hover effects.

Category: Navigation Components

Dark Mode Support: Yes

Production Classes: .sidebar, .sidebar-nav, .sidebar-nav-item

Main Content Area

This demonstrates how the sidebar works alongside main content. The sidebar has a sticky position that stays in view during scrolling.

States

  • default - Normal sidebar link with standard styling
  • hover - Link highlights with background color and subtle left border
  • active - Current page with blue accent border and bold font weight
  • sticky - Sidebar remains fixed at top: 80px during scroll

Variants

  • .sidebar - Main sidebar container with sticky positioning
  • .sidebar-heading - Section heading within sidebar (uppercase, small text)
  • .sidebar-nav - Navigation list container
  • .sidebar-nav-item - Individual navigation link item
  • .sidebar-nav-item-active - Active/current section indicator
  • .sidebar-divider - Horizontal divider between sections
  • .sidebar-nav-submenu - Nested submenu for hierarchical navigation

HTML Example - Basic Sidebar

<nav class="sidebar">
  <div class="sidebar-heading">Navigation</div>
  <ul class="sidebar-nav">
    <li><a href="#home" class="sidebar-nav-item">Home</a></li>
    <li><a href="#projects" class="sidebar-nav-item sidebar-nav-item-active">Projects</a></li>
    <li><a href="#settings" class="sidebar-nav-item">Settings</a></li>
  </ul>
</nav>

CSS Classes

Class Description
.sidebar Main sidebar container with sticky positioning and scrollbar
.sidebar-heading Section heading (uppercase, small, bold)
.sidebar-nav Navigation list container (removes default list styles)
.sidebar-nav-item Individual navigation link with border-left transitions
.sidebar-nav-item-active Active/current section indicator with bold font
.sidebar-nav-item.active Alternative active class syntax
.sidebar-divider Horizontal divider line between sections
.sidebar-nav-toggle Collapsible section toggle button
.sidebar-nav-submenu Nested submenu container for hierarchical navigation

Responsive Behavior

  • Desktop (>992px) - Sticky sidebar with full height and scroll
  • Tablet (768-992px) - Static position, horizontal layout possible
  • Mobile (<768px) - Reduced padding, smaller font size, border-bottom instead of border-right
