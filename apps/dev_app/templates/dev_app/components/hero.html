Hero

Hero section component for prominent page introductions

{% include 'dev_app/design_partial/sidebar.html' %}

Overview

Hero sections are large banner areas at the top of pages designed to grab attention with a headline, subheading, and call-to-action buttons.

Category: Layout Components

Dark Mode Support: Yes

Production Classes: .hero, .hero-content, .hero-title, .hero-subtitle, .hero-description, .hero-buttons

Variants

  • .hero - Standard hero with light background
  • .hero-silverish-ai-light - Clean, professional light appearance
  • .hero-silverish-ai-soft - Gentle gradient transitions
  • .hero-silverish-ai-radial - Radial gradient with focal point
  • .hero-silverish-ai-subtle - Very gentle, imperceptible gradient
  • .hero-silverish-ai - Standard silverish AI gradient
  • .hero-bg-image - Background image with dark overlay
  • .hero-angled - Angled bottom edge
  • .hero-split - Split layout with text and image
  • .hero-scholar - Scholar module accent
  • .hero-writer - Writer module accent
  • .hero-code - Code module accent
  • .hero-viz - Viz module accent

States

  • default - Standard appearance with centered content
  • bg-image - Background image with dark overlay for text contrast
  • responsive - Font sizes and layout adjust on mobile devices

      <a href="#" class="btn btn-primary">Get Started</a>
