Card

Flexible content container with optional header, body, and footer sections

{% include 'dev_app/design_partial/sidebar.html' %}

Overview

Cards are flexible content containers that group related information together. They can include headers, body content, and footer actions.

Category: Layout Components

Dark Mode Support: Yes

Production Class: .card

Variants

  • .card - Base card with border and shadow
  • .card-primary - Primary color variant
  • .card-secondary - Secondary color variant
  • .card-accent - Accent color variant
  • .card-clickable - Interactive with hover and active states
  • .card-hover - Hover effect only (no active state)
  • .card--feature - Feature showcase with checkmarks
  • .card--repo - Repository listing with lift hover
  • .card--module - Product/module showcase
  • .card--requirement - System requirement display
  • .card--security - Security feature with left border

HTML Example

<!-- Basic Card -->
<div class="card">
  <div class="card-header">
    <h5 class="card-title">Card Title</h5>
  </div>
  <div class="card-body">
    <p class="card-text">Card content goes here.</p>
  </div>
</div>

<!-- Card with Footer -->
<div class="card">
  <div class="card-header">
    <h5 class="card-title">Actions</h5>
  </div>
  <div class="card-body">
    <p class="card-text">Content with action buttons.</p>
  </div>
  <div class="card-footer">
    <button class="btn btn-primary">Save</button>
  </div>
</div>

<!-- Card with Image -->
<div class="card">
  <img src="image.jpg" class="card-img-top" alt="Card image">
  <div class="card-body">
    <h5 class="card-title">Image Card</h5>
    <p class="card-text">Text below image.</p>
  </div>
</div>

<!-- Clickable Card -->
<div class="card card-clickable">
  <div class="card-body">
    <h5 class="card-title">Interactive</h5>
    <p class="card-text">Hover and click me!</p>
  </div>
</div>

CSS Classes

Class Description
.card Base card container (required)
.card-header Optional header section with background
.card-body Main content area with padding
.card-footer Optional footer section with background
.card-title Card title (h5-sized, bold)
.card-subtitle Card subtitle (smaller, muted)
.card-text Paragraph text with spacing
.card-link Inline link with spacing
.card-img-top Image at top of card (rounded corners)
.card-img-bottom Image at bottom of card (rounded corners)
.card-img-overlay Overlay text on card image
.card-primary Primary color card variant
.card-secondary Secondary color card variant
.card-accent Accent color card variant
.card-clickable Interactive card with hover and active effects
.card-hover Hover effect only (no active state)
.card-deck Flex container for equal-width cards
.card-group Grouped cards with connected borders
.card-columns Masonry-style card columns (3 columns)
.card--feature Feature card with checkmarks
.card--repo Repository listing card
.card--module Product/module showcase card
.card--requirement System requirement card
.card--security Security feature card with left border
