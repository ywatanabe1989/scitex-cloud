Card

Container component for grouping related content

Overview

Category: Container & Layout

Dark mode: Supported

Cards are flexible content containers that group related information together. They can include headers, body content, and footer actions.

Component Demo

This is a basic card with header and body. It provides a simple container for content organization.

Card with Actions

Includes footer actions

This card demonstrates how to include action buttons in the footer section.

With shadow elevation

This card uses shadow elevation instead of borders to create depth and visual hierarchy.

Filled Card

With background fill

This card variant uses a filled background color for better visual distinction.

A simplified card variant with minimal padding for dense layouts.

Variants & States

  • Basic: Standard card with border
  • Elevated: Card with shadow elevation
  • Filled: Card with background fill
  • With Header: Card with header section
  • With Footer: Card with footer section for actions
  • Compact: Minimalist card variant
  • Hover State: Elevation and transform on hover

HTML Code Example

<!-- Basic Card -->
<div class="card">
  <div class="card-header">
    <h3 class="card-title">Card Title</h3>
  </div>
  <div class="card-body">
    <p>Card content goes here</p>
  </div>
</div>

<!-- Card with Footer -->
<div class="card">
  <div class="card-header">
    <h3 class="card-title">Actions</h3>
  </div>
  <div class="card-body">
    <p>Content with action buttons</p>
  </div>
  <div class="card-footer">
    <button class="btn btn-primary">Save</button>
  </div>
</div>

<!-- Elevated Card -->
<div class="card card-elevated">
  <div class="card-body">
    <p>Elevated card with shadow</p>
