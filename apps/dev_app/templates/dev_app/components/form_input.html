Overview

Form input fields are fundamental form elements for capturing user text input. They support various states including focused, disabled, and validation states (success/error).

Category: Form and Selection

Dark Mode Support: Yes

States

  • default - Default input state
  • focus - Input is focused with blue border and shadow
  • success - Validation success with green border
  • error - Validation error with red border
  • disabled - Input is disabled and cannot be edited
