Form Input

Text input fields with various states, sizes, and validation states

{% include 'dev_app/design_partial/sidebar.html' %}

Overview

Form input fields are fundamental form elements for capturing user text input. They support various states including focused, disabled, and validation states (success/error/warning).

Category: Form and Selection

Dark Mode Support: Yes

Production Classes: .form-control, .form-group, .form-label

States

  • default - Normal interactive state
  • focus - Highlighted border and shadow when focused
  • valid - Valid input state with green border and checkmark
  • invalid - Invalid input state with red border and error icon
  • warning - Warning state with yellow border and warning icon
  • disabled - Non-interactive disabled state
  • readonly - Read-only state (not editable)

Sizes

  • .form-control-sm - Small input field
  • .form-control - Default size
  • .form-control-lg - Large input field

HTML Example

<div class="form-group">
  <label class="form-label" for="email">Email Address</label>
  <input type="email" id="email" class="form-control" placeholder="you@example.com">
  <div class="form-text">We'll never share your email</div>
</div>

<!-- Valid input -->
<div class="form-group">
  <label class="form-label" for="username">Username</label>
  <input type="text" id="username" class="form-control is-valid" value="john_doe">
  <div class="valid-feedback">Looks good!</div>
</div>

<!-- Invalid input -->
<div class="form-group">
  <label class="form-label" for="password">Password</label>
  <input type="password" id="password" class="form-control is-invalid">
  <div class="invalid-feedback">Password is required</div>
</div>

<!-- Textarea -->
<div class="form-group">
  <label class="form-label" for="message">Message</label>
  <textarea id="message" class="form-control" rows="4"></textarea>
</div>

CSS Classes

Class Description
.form-control Base input/textarea/select class
.form-control-sm Small size variant
.form-control-lg Large size variant
.form-group Container for label + input with spacing
.form-label Label styling for form fields
.form-text Helper text below input
.is-valid Valid state with green border
.is-invalid Invalid state with red border
.is-warning Warning state with yellow border
.valid-feedback Success message (shown with .is-valid)
.invalid-feedback Error message (shown with .is-invalid)
.warning-feedback Warning message (shown with .is-warning)
