Overview

Module icons provide consistent visual identity across the SciTeX ecosystem. Each module has a distinctive icon available in both SVG (professional) and emoji (friendly) variants, supporting dark mode and multiple size options.

Category: Visual Identity

Dark Mode Support: Yes

Production Classes: .module-icon, .emoji-icon, .nav-icon

CSS Classes

Class Description
.module-icons-grid Grid layout for displaying SVG module icon cards
.icon-showcase Individual SVG module icon card container
.emoji-icons-grid Grid layout for displaying emoji icon cards
.emoji-showcase Individual emoji icon card container
.module-icon Container for SVG module icons (display: inline-block)
.emoji-icon Container for emoji module icons (3.5rem font size)
.nav-icon Navigation header icon (20px, with opacity transitions)
.nav-icon-svg SVG navigation icon with theme color support
.module-icon-hero Hero section icons (64px)
.module-icon-card Card icons (48px)
.module-icon-list List item icons (32px)
.module-icon-inline Inline text icons (24px)
