SciTeX Color System - Minimal Semantic Tokens

Our simple color system uses theme-aware semantic tokens that automatically adapt to light and dark modes. All token values are mapped to ensure proper contrast and readability.

Simple Approach: Two-tier system with primitive tokens (_prefix, internal only) and semantic tokens (public API). Use semantic tokens in your components.

Text Colors (Semantic)

These adapt automatically based on theme (light/dark mode).

--text-primary

--text-inverse

  • Use Semantic Tokens: Always use --text-primary, --bg-page, etc. in your components
  • Primitives are Internal: Tokens with _ prefix are only used in colors.css
  • Theme Agnostic: Same semantic token names work in both light and dark modes automatically
  • Minimal System: Only use what you need - text, background, border, and status colors
