Theme System

/static/shared/css/theme.css

Modular CSS variable system with light/dark theme support using data-theme attribute.

Light Mode

<html lang="en" data-theme="light">
</html>

Text: var(--color-fg-default)

Muted: var(--color-fg-muted)

Dark Mode

<html lang="en" data-theme="dark">
</html>

Text: var(--white)

Muted: var(--white) with opacity
