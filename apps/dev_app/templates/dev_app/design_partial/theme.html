Theme System

/static/css/theme.css

SciTeX uses a modular CSS variable system with light/dark theme support. The main theme file is /static/css/theme.css with support files in /static/css/common/.

Modular CSS Structure

Main Theme: /static/css/theme.css (Light & Dark mode with data-theme attribute)

Variables Entry Point: /static/css/common/variables.css

Variables are split into focused files:

  • colors.css - All color definitions (brand, status, semantic)
  • typography-vars.css - Font families, sizes, line heights
  • spacing.css - Spacing scale and layout widths
  • effects.css - Borders, shadows, transitions
  • z-index.css - Z-index scale

Light & Dark Mode Support

SciTeX supports light and dark themes using the data-theme attribute on the <html> tag.

Light Mode (Default)

          <html lang="en" data-theme="light">
          <!-- Light mode active -->
          </html>

Text (--color-fg-default): Uses actual theme variable

Muted (--color-fg-muted): Uses actual theme variable

Canvas subtle (--color-canvas-subtle)

Dark Mode

          <html lang="en" data-theme="dark">
          <!-- Dark mode active -->
          </html>

Text (--color-fg-default): Pure white in dark mode

Muted (--color-fg-muted): White in dark mode

Canvas subtle: --scitex-color-01

Important: Always use theme variables (var(--color-fg-default)) instead of hardcoded colors to ensure proper contrast in both modes.

Dark Mode Update: Text colors now use pure white (var(--white)) for maximum visibility and contrast in dark mode.
