Icon Color System

Icons use a warm yellowish color scheme that adapts between light and dark modes.

Light Mode
rgb(184, 164, 130) Default
rgb(204, 184, 150) Hover/Active 
.nav-icon-fa {
  color: rgb(184, 164, 130);
  opacity: 0.85;
}

.header-nav-item:hover .nav-icon-fa {
  color: rgb(204, 184, 150);
  opacity: 1;
}
CSS Filters (for IMG icons)

Applied to .nav-icon class for image-based icons

/* Default state */
.nav-icon {
  filter: brightness(1) saturate(1)
          sepia(0.5) hue-rotate(10deg)
          opacity(0.85);
}

/* Hover/Active state */
.header-nav-item:hover .nav-icon {
  filter: brightness(1.1) saturate(1.2)
          sepia(0.6) hue-rotate(15deg)
          opacity(1);
}

Best Practices

  • ‚úì Use FontAwesome for consistency - All navigation icons use FontAwesome 6.0.0 for visual consistency
  • ‚úì Apply proper CSS classes - Use .nav-icon-fa for FontAwesome icons, .nav-icon-svg for SVG icons
  • ‚úì Maintain alignment - Ensure icons and text are vertically centered using flexbox
  • ‚úì Use standard sizes - Navigation icons should be 16px (font-size) or 20px (container height)
  • ‚úó Don't mix icon systems - Avoid mixing FontAwesome, SVG, and image icons in the same navigation
  • ‚úó Don't hardcode colors - Use CSS classes and variables instead of inline styles
