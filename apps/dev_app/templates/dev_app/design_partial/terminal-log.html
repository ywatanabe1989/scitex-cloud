JavaScript Integration

Append messages programmatically and auto-scroll to bottom

// Append log message and auto-scroll
function appendLog(logElement, message, type = 'info') {
  const span = document.createElement('span');
  span.className = `terminal-log__${type}`;
  span.textContent = message;

  logElement.appendChild(document.createTextNode('\n'));
  logElement.appendChild(span);

  // Auto-scroll to bottom
  logElement.scrollTop = logElement.scrollHeight;
}

// Usage
const log = document.querySelector('.terminal-log');
appendLog(log, '✓ Task completed', 'success');
appendLog(log, '✗ Error occurred', 'error');
appendLog(log, '⚠ Warning message', 'warning');
appendLog(log, 'ℹ Information', 'info');

Best Practices

  • Use semantic message types - Apply appropriate classes for success, error, warning, and info messages
  • Auto-scroll - Always scroll to bottom when appending new messages for real-time monitoring
  • Size appropriately - Use compact for simple tasks, large for detailed logging
  • Add timestamps - Include timestamps for debugging and tracking
  • Limit history - For long-running processes, consider limiting log history to prevent memory issues
  • Provide controls - Add copy and clear buttons for user convenience
