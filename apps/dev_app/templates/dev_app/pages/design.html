Usage Guidelines

The following guidelines ensure consistent implementation of the design system:

Do:

  • Use consistent spacing variables
  • Follow the established color palette
  • Use the component system for UI elements
  • Maintain consistent typography hierarchies
  • Design for accessibility and responsive layouts

Don't:

  • Create custom colors outside the palette
  • Use inconsistent spacing
  • Create custom component variants without documentation
  • Use fonts not specified in the design system
  • Ignore responsive design principles

Hero Gradient Usage Guidelines

When to Use Each Variant:

  • Light: Writer, content-focused pages, softer UI - primary choice for most applications
  • Radial: Featured content, focal highlights, innovation pages, central call-to-actions
  • Soft: Collaborative spaces, community areas, research portals, team-focused features
  • Subtle: Minimal UI, background sections, secondary content, unobtrusive interfaces

Recommended Applications:

  • Scholar: Light or Soft
  • Writer: Light (primary) or Soft
  • Viz: Radial or Light
  • Code: Subtle or Light
  • Landing Page: Radial or Light

Implementation:

CSS File:
/static/common/css/hero-gradients.css
Basic Usage:
<div class="hero-section hero-silverish-ai">
With Module Class:
<div class="hero-section hero-silverish-ai hero-scholar">
Available Classes:
.hero-silverish-ai-light
.hero-silverish-ai-radial
.hero-silverish-ai-soft
.hero-silverish-ai-subtle

Design Principles:

  • Scientific Sophistication: Metallic progression conveys precision and technological advancement
  • AI Emphasis: Silverish tones suggest AI sophistication and innovation
  • Brand Consistency: All variants use SciTeX color palette
  • Accessibility: Text shadows and color choices ensure readability
  • Visual Hierarchy: Different variants for different content priorities
  • Responsive Design: All gradients work across screen sizes

Color Psychology:

  • Bluish Gray: Trust, stability, intelligence
  • Metallic Progression: Innovation, quality, precision
  • Gradient Flow: Progress, advancement, evolution
