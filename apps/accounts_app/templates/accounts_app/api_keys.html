How to use API keys

Include your API key in the Authorization header of your HTTP requests:

curl -H "Authorization: Bearer scitex_your_key_here" \
     https://scitex.ai/api/projects/

Or in Python:

import requests

headers = {"Authorization": "Bearer scitex_your_key_here"}
response = requests.get("https://scitex.ai/api/projects/", headers=headers)
