SciTeX Scholar Demo

AI-powered literature discovery and analysis

  • Multi-Source Search
    Concurrent search across CrossRef, PubMed, OpenAlex, Semantic Scholar, and arXiv
  • Metadata Enrichment
    Automatic retrieval of abstract, citation counts, and journal impact factor
  • PDF Downloads
    PDF download with institutional authentication support. Extraction of text, tables, and figures
  • Local Library
    Manage and search your local PDF collection. Compatible with Zotero.
  • Exporting Formats
    Export to BibTeX, RIS, JSON, and Markdown formats
  • LaTeX Integration
    LaTeX compilation with reference management, style changes, and words counts
  • Manuscript Types
    Templates ready for manuscripts, supplementary materials, and revision letters
  • AI Applicability
    Optimized structure for LLM assistants. Automatic differences with visual markup
  • Bibliography Management
    Fact-based reference insertion with Scholar module
  • Version Control
    Git-based workflow with manuscript versioning
